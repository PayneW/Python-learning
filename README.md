- 常用 Linux 命令的基本使用

    | 序号 | 命令 | 对应英文 | 作用 |
    |:----:|:----:|:----:|:----:|
    | 01 | ls | list | 查看当前文件夹下的内容 |
    | 02 | pwd | print work directory(打印工作目录) | 查看当前所在文件夹 |
    | 03 | cd [目录名] | change directory | 切换文件夹 |
    | 04 | touch [文件名] | touch | 如果文件不存在，新建文件 |
    | 05 | mkdir [目录名] | make directory | 创面目录 |
    | 06 | rm [文件名] | remove | 删除指定的文件名 |
    | 07 | clear | clear | 清屏 |

- 在终端中放大/缩小字体:
    + 放大: `ctrl + shift + =`
    + 缩小: `ctrl + -`
